Donasi untuk Ruby on Rails Indonesia

Untuk saat ini Rails Indonesia hanya menerima donasi berupa cloud computing architecture misalnya seperti:

  • Dedicated Server / VPS, seperti:
    • Provider Dedicated Server Indonesia
    • Digital Ocean
    • Vultr
    • Atau yang lainnya
  • Cloud Computing Services, seperti:
    • Heroku
    • Amazon Web Services (AWS)
    • Google Cloud Platform (GCP)
    • Alibaba Cloud
    • Azure
    • Atau yang lainnya

Yang dapat bertahan dalam jangka waktu panjang.

Hubungi tim inti

Dari manapun (organisasi, perusahaan ataupun perorangan) yang ingin memberikan donasi untuk kelangsungan hidup komunitas Rails Indonesia dapat menghubungu tim inti Rails Indonesia dibawah ini:

Forum Diskusi

Telegram

Dengan begitu Rails Indonesia berhak menyantumkan dukungan berupa identitas biasanya logo dari siapapun yang telah memberikan donasi di halaman situs web Rails Indonesia serta akan menambahkan di cover setiap Raild Indonesia mengadakan Event.
